# 은수 지갑 (silverwater_wallet)

영수증 사진에서 금액/날짜를 추출해 로컬에 저장하고, 날짜별/월별 지출을 확인하는 개인 가계부 웹앱입니다.

## 배포 주소

- <https://snowpero.github.io/silverwater_wallet/>

## 주요 기능

- 영수증 이미지 업로드 후 로컬 OCR(Tesseract.js) 추출
- 추출된 결제일/총 금액 자동 입력 + 수동 수정 저장
- 날짜별 내역 그룹 보기
- 결제일 기준 일평균 사용금액 계산
- 월별 사용금액 그래프
- 이번달/지난달 총합 및 증감 비교
- 등록 내역 삭제
- 등록 내역 상호명 인라인 수정

## 사용 방법

1. 페이지 접속 후 `영수증 이미지`를 선택합니다.
2. `OCR 추출` 버튼을 눌러 자동 추출을 실행합니다.
3. 결제일/총 금액/상호명을 확인 또는 수정합니다.
4. `저장` 버튼으로 로컬 DB(IndexedDB)에 저장합니다.
5. 저장된 내역은 날짜별 목록, 요약 카드, 월별 그래프에 즉시 반영됩니다.

## 배포 버전 표시

- 화면 하단에 `배포 YYYY.MM.DD | HH:MM KST | 커밋해시` 형태로 노출됩니다.
- `최신 버전 불러오기` 버튼은 새 배포가 감지된 경우에만 표시됩니다.

## 기술 스택

- HTML, CSS, JavaScript (ES Modules)
- IndexedDB
- Tesseract.js
